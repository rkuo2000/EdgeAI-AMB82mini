2. 開發環境使用介紹

安裝 Arduino IDE 2.3.10

下載 Arduino IDE

偏好設定 (Preferences)

於 Arduino IDE 偏好設定中加入以下 Board Manager URL：

HUB8735 Ultrahttps://raw.githubusercontent.com/ideashatch/HUB-8735/main/amebapro2_arduino/Arduino_package/ideasHatch.json
AMB82-mini (main)https://github.com/Ameba-AIoT/ameba-arduino-pro2/raw/main/Arduino_package/package_realtek_amebapro2_index.json
AMB82-mini (dev)https://github.com/Ameba-AIoT/ameba-arduino-pro2/raw/dev/Arduino_package/package_realtek_amebapro2_early_index.json

選擇開發板

  1. Tools > Board Manager > 搜尋 AMB82
  2. 安裝 Realtek Ameba Boards 4.1.0-build20260213
  3. Serial-monitor 設定為 115200 baud

Getting Started

  1. 將 AMB82-mini 用 MicroUSB 線連接至電腦 USB port
  2. 確認 UART com port (Ubuntu: sudo chown usrname /dev/ttyUSB0)
  3. 進入燒錄模式：按 UART_DOWNLOAD → 按 RESET → 放開 RESET → 放開 UART_DOWNLOAD
  4. 於 Arduino IDE 按下 Upload 燒錄程式碼
